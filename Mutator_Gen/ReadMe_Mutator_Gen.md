# Mutator_Gen Directory Overview
This top-level folder tracks the historical generation, testing, and implementation of our various prompt mutation engines before they were refactored completely into the modern FastAPI pipeline.

### Contained Artifacts:
- **`G/`**: A nested subdirectory which houses the original standalone scripts for building postgres tables, extracting dimensional genomes, and automatically feeding adversarial prompts into local evaluation pipelines.
- **Various Jupyter Notebook Checkpoints**: Stored test explorations from initial R&D.
